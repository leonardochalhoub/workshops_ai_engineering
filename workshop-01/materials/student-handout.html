② O que é Layer Zero

Layer Zero é a virada de mentalidade que você já está estudando: a parte difícil de fazer software não é mais a sintaxe. É saber o que construir e por quê. Com a IA como equipe de palco, uma pessoa lança o que antes exigia um time. Venha curioso(a), não perfeito(a).

③ Checklist de preparo (antes da Semana 2)

Você não precisa de nada disso para o Workshop 1 — mas faça antes da Semana 2 para começarmos a construir. Tudo grátis.

  • GitHub — crie uma conta grátis em github.com/signup
  • Node.js LTS — nodejs.org (o botão “LTS”)
  • VS Code — code.visualstudio.com
  • Claude Code — nosso par de IA (o Leo compartilha o passo de instalação)
  • Supabase — conta grátis em supabase.com
  • Vercel — conta grátis em vercel.com (entre com o GitHub)
